# Intel 8080 Emulator
<img src = "/screenshots/SpaceInvaders.gif?raw=true">
Exactly what it sounds like.

## Build
`mkdir build && cd build`  
`cmake -B. --config Release ..`  
`make`

## Run
### Linux
`./intel-8080-tests`

### Windows
`intel-8080-tests.exe`
